About us

The Project

The USI help system is the result of a project for the CS 478 Software Development class taught by Dr. Bruce Mabis.

The System

The USI help system is a generalized help system focused on computer hardware troubleshooting. To use the system, pick a hardware category from the menu, then pick the specific model you wish to know about. If the database does not have the answer you need, then you can submit more information on that page or contact the developers for further help.
